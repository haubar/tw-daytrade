# 新手教學：怎麼看懂「阿韭衝衝衝判斷器」

這份文件是給第一次使用這個工具的人看的，會照這個順序講：
1. 這是什麼工具，能幫你做什麼
2. 畫面上每個東西是什麼意思
3. 怎麼使用拖曳式篩選與價格帶提示
4. 四個因子分別在算什麼、為什麼要看這個
5. 分數是怎麼加總出來的（進階，想知道原理才需要看這節）
6. 資料多新鮮、什麼時候該相信它
7. 常見問題

看完這份文件，你應該要能自己判斷「這檔股票為什麼會出現在觀察榜上」，而不是只看到一個分數就照單全收。

---

## 0. 先講清楚：這是輔助工具，不是明牌機

- 這個工具做的事，是把台股全市場（上市＋上櫃）的公開資料抓下來、算幾個量化指標、排序出來給你看，**不會**告訴你「一定要買」或「一定要賣」
- 出現在觀察榜上，代表這檔股票「今天的某些數據特徵比較突出」，不代表它接下來會漲或跌
- 當沖有資格限制、稅務規則、風險限制，這些工具都不會幫你處理，請自己確認
- 系統本身也有已知的限制（例如暖機期間資料不完整、大盤漲跌幅是估計值），底下會逐一說明，不要照單全收

## 1. 畫面上每個東西是什麼意思

### 頂部狀態列

| 項目 | 意思 |
|---|---|
| 資料時間 | 這次結果是什麼時候算出來的（不是即時的，是最近一次排程/手動觸發的時間） |
| 大盤漲跌幅（近似） | **不是**真正的加權指數漲跌幅，是用全市場所有股票的成交值加權平均漲跌幅去估計出來的替代值。之後如果接上真正的大盤指數，這裡會更準 |
| 候選檔數 | 排除掉「歷史資料還不夠」或「今天沒有交易」的股票之後，實際參與排名的股票總數 |

### 多方觀察榜 / 空方觀察榜

畫面會分成左右（或手機上下）兩欄：

- **多方觀察榜**（紅色）：偏多訊號的股票，排名由強到弱
- **空方觀察榜**（綠色）：偏空訊號的股票，排名由強到弱

> ⚠️ **紅漲綠跌，跟美股相反！** 這是台股的慣例，如果你習慣看美股（美股是紅跌綠漲），第一次看可能會覺得顏色「怪怪的」，這是刻意設計成符合台灣看盤習慣，不是顏色用錯了。

每一列（每檔股票）由左到右分別是：

1. **排名**（01、02、03...）
2. **股票代號 + 名稱 + 市場標籤**（上市／上櫃）
3. **收盤價 + 漲跌幅**
4. **因子解剖條**（見下一段）
5. **綜合分數**（最右邊的數字，就是排名依據，分數越高代表訊號越強）

### 因子解剖條——這是這個工具的核心視覺

每一列右邊那條彩色長條，把「這檔股票為什麼上榜」拆開來看，不是只丟一個分數給你、要你自己相信黑盒子：

| 顏色 | 代表哪個因子 |
|---|---|
| 🟡 金色 | 量能異常的貢獻 |
| 🔴／🟢（跟該面板同色） | 跳空幅度的貢獻 |
| 🟣 靛色 | 相對大盤強弱勢的貢獻 |
| 🔵 青色 | 三大法人買賣超的貢獻 |

每一段的**長度**代表這個因子對總分的貢獻有多大。舉例：如果一檔股票的解剖條大部分是金色，代表它主要是「量能異常放大」把分數推高的，跳空跟法人買賣超的貢獻不大；如果是青色佔大部分，代表主要是法人資金介入的訊號在推動。

**這對你的用處**：兩檔股票總分可能一樣，但背後原因完全不同——一檔可能是單純爆量、一檔可能是法人大買，你可以依照自己的交易邏輯，選擇比較信任哪一種訊號組成的股票，而不是只看總分數字。

## 2. 怎麼使用篩選與價格帶提示

篩選面板只會即時改變「你看到哪些股票」，**不會重算分數或改變排行榜原本的順序**。因此可以安心拖曳條件，比較不同交易偏好下還有哪些標的符合。

### 拖曳式條件

- **股價範圍**：兩個滑桿分別設定最低與最高價；也可直接點選下方的價格帶。
- **最小成交量**：以台股慣用的「張」設定為不限、100、500、1,000、5,000、10,000 張以上。
- **最小漲跌幅度**：設定為不限、1%、3%、5%、7%、10%。系統取絕對值，所以多方看漲幅、空方看跌幅，使用同一個門檻。
- **清除篩選**：會清空以上三項由你設定的條件，回到完整的當沖參考名單。

> 千元以上股票是固定排除的當沖參考規則，並非暫時篩選條件；所以按「清除篩選」後也不會重新出現。

### 價格帶與「參考略過檔數」

每個價格帶按鈕與榜單內的提示，會顯示對應的「參考略過 N 檔」。這是使用者提供的操作參考：先略過該數量的候選檔，再自行依盤中型態、流動性與風險控管判斷；它**不會自動刪除股票，也不保證獲利**。

| 股價區間（元） | 參考略過檔數 |
|---|---:|
| 370–500 | 3 |
| 184–370 | 2 |
| 100–184 | 1 |
| 75–100 | 3 |
| 50–74 | 2 |
| 38–50 | 3 |
| 18–37 | 2 |
| 11–18 | 1 |
| 3.7–10 | 2 |
| 3.6 以下 | 1 |

實際股價若落在 74.5、37.5 等正常跳動價格，系統會歸到相鄰價格帶，不會漏掉。500～999 元仍可顯示，但因原始規則沒有提供略過檔數，系統不會自行猜測提示。

## 3. 四個因子分別在算什麼

### 因子 1：量能異常（權重 30%）

**公式**：當日成交量 ÷ 過去 3 個交易日均量

**白話**：今天的成交量是平常的幾倍？

**直覺**：想像一檔股票平常一天成交 1,000 張，今天忽然成交 5,000 張，是平常的 5 倍——不管是漲是跌，這種「量能異常放大」通常代表有特定原因在驅動這檔股票（消息面、法人進出、主力介入等），是短線交易者常用的第一層篩選條件：先找「今天特別熱門」的股票，再看它熱門的原因是什麼。

### 因子 2：跳空幅度（權重 20%）

**公式**：(今天開盤價 − 昨天收盤價) ÷ 昨天收盤價 × 100%

**白話**：開盤的時候，價格比昨天收盤跳高還是跳低多少？

**直覺**：開盤跳空往往反映「隔夜/盤前的情緒與消息」，跳空幅度越大（不管方向），代表市場對這檔股票的預期有明顯轉變。當沖交易者通常會特別留意跳空缺口附近的價格行為（例如「補缺口」或「缺口延續」這類經典型態）。

### 因子 3：相對大盤強弱勢（權重 20%）

**公式**：個股當日漲跌幅 − 大盤（近似）當日漲跌幅

**白話**：這檔股票今天的表現，比大盤好還是差？

**直覺**：如果大盤大漲 5%，個股也漲 5%，那只是「跟著盤動」而已，不算真正強勢；但如果大盤只漲 1%，這檔股票卻漲了 8%，代表它「相對大盤」真的特別強——這才是這個因子想抓的訊號，用意是排除掉「因為大盤好、什麼股票都在漲」的雜訊，找出真正有自己動能的股票。

### 因子 4：三大法人買賣超比例（權重 30%）

**公式**：（外資 + 投信 + 自營商當日買賣超股數）÷ 當日成交量 × 100%

**白話**：今天的成交量裡，有多大比例是三大法人（外資、投信、自營商）在買/賣？

**直覺**：法人資金通常比散戶擁有更多資訊或更具規模，如果一檔股票今天的成交量裡有很高比例是法人買超，代表「相對專業的資金」可能正在進場，這是一個籌碼面的訊號。

> **補充說明**：這裡原本規劃放的是「隔日沖分點買賣超」（特定券商分點的進出資料），但那個資料源需要繞過官方的圖形驗證碼保護才能自動抓取，我們不做這件事（這是網站刻意設置的防護機制，繞過它不是應該做的事）。改用三大法人買賣超資料替代——雖然不是真正的分點級資料，但同樣屬於「大額資金動向」的訊號，而且是完全合法、免費公開的官方資料。

## 4. 分數是怎麼加總出來的（進階）

如果你只是想知道「這個工具幫我做了什麼」，看到這裡就夠了。如果你想知道分數的計算細節，繼續看下去。

### 第一步：把每個因子換算成「百分位分數」

四個因子的單位完全不一樣（量能是「倍數」、跳空是「百分比」、法人買賣超是「佔成交量比例」），沒辦法直接加在一起比較。所以系統會先把每個因子在「全部候選股票」裡做排名，換算成 0～100 的百分位分數：

- 某檔股票的量能異常倍數，在全部候選股票裡排第幾名（由小到大），就換算成對應的 0～100 分數
- 跳空幅度、相對強弱、法人買賣超比例，也都各自做一次同樣的換算

這樣不管原始單位差多少，四個因子都變成同一個量級（0～100），才能公平地加權合併。

### 第二步：依權重加總

```
綜合分數 = 量能異常百分位 × 30% + 跳空幅度百分位 × 20% + 相對強弱百分位 × 20% + 法人買賣超百分位 × 30%
```

理論最高分是 100（四個因子都排在候選股票裡的最強勢那一端）。

### 第三步：多方/空方怎麼分別排序

- **多方觀察榜**：四個因子都是「原始方向」直接排序——量能異常一定是正向貢獻（爆量永遠是訊號強，不管漲跌），跳空、相對強弱、法人買賣超則是「越正（越強勢/越買超）分數越高」
- **空方觀察榜**：量能異常的計算方式不變（爆量還是爆量，不會因為要找空方就變成「量縮」），但跳空、相對強弱、法人買賣超這三個「有方向性」的因子會反過來計算——越是跳空向下、越弱勢、法人賣超越多，分數才會越高

> 這裡有個容易搞錯的地方：**空方觀察榜不是「把多方分數低的股票排前面」**。如果只是把多方分數倒過來排序，會把「量能很低、沒什麼動作的平淡股」誤判成空方訊號（因為它們量能因子貢獻低，倒過來排序反而分數高），但那其實只是平淡，不是「主動走弱」。真正的空方訊號應該是「爆量＋下跌」，量能異常的方向不能反轉，只有跳空/相對強弱/法人買賣超的方向要反轉。

## 5. 資料多新鮮？什麼時候該相信它

- **每天台灣時間下午 2 點後才會更新當天資料**：台股 13:30 收盤，太早查詢可能拿到還沒最終確認的盤後資料，所以系統會等到下午 2 點後才把當天資料當作正式收盤資料存起來。如果你在下午 2 點前看到結果，代表用的可能是前一個交易日的資料
- **週末不會更新**：星期六日不是交易日，系統不會產生新的資料
- **量能異常因子需要「暖機」**：這個因子要比較「今天 vs 過去 3 天」的成交量，剛開始使用（或資料庫剛清空）的前幾天，因為還沒累積到 3 天的歷史資料，這個因子會先是中性值，其他三個因子不受影響，正常運作

## 6. 常見問題

**Q：觀察榜是空的，是不是壞了？**
A：不一定。可能是（1）還在暖機期，歷史資料不夠 (2) 剛好在交易時段中還沒到下午 2 點的更新時間 (3) 今天剛好是非交易日。畫面上會有清楚的狀態提示告訴你目前是什麼狀況。

**Q：畫面上寫「範例資料」是什麼意思？**
A：代表目前顯示的不是真實行情，是示範用的假資料，通常是因為還沒接上真實後端，或後端還沒有任何結果可以讀取。

**Q：綜合分數高，是不是代表這檔股票一定會照我想的方向走？**
A：不是。分數只代表「這幾個量化特徵在今天全市場裡相對突出」，不是漲跌預測。因子解剖條可以幫你判斷這個訊號的組成是什麼，但最終判斷跟風險承擔還是在你自己身上。

**Q：多方觀察榜跟空方觀察榜可以同時看嗎？**
A：可以，兩份榜單是同時產生的、互相獨立，分別代表當天全市場裡「偏多訊號最強」跟「偏空訊號最強」的股票。

**Q：大盤漲跌幅為什麼是「近似」，不能用真的大盤指數嗎？**
A：目前用的是全市場成交值加權平均漲跌幅去估計，跟真正的加權指數（TAIEX）會有落差，但方向跟大致幅度應該接近。這是目前系統的已知限制，之後有機會會接上真正的大盤指數資料。
